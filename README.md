# The Backend of KING Pacman site

## Release date
May 27, 2022

## Environment
- `Node.js v14.17.0`
- `MySQL v8.0.27`

## Stack
- **Framework:** `Express.js v4.17.3`
- **Database:** `MySQL`

## How to run the game.
1. Please open terminal window in the root directory.
2. Please run command `npm run start` in it.
